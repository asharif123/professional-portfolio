Room Finder App

On RoomFinder, users can explore housing options nationwide using Google API and Roomster API.

The user enters the city, state of where they are looking and their max budget.

The user can also see and clear their previous search history.

Technologies used: HTML, CSS, Javascript, Bootstrap, Responsive Web Design, Sequelize, Express, Node.js, Tailwind
